# -Retail-Store-Sales-Insights-and-Prediction-Model
Retail sales analysis project using Python and Power BI to uncover trends, customer behavior, and business insights. Includes data cleaning, EDA, and an interactive dashboard showing sales performance, regional trends, and top products for better decision-making.

## Project Overview
This project analyzes retail sales data using Python and Power BI to uncover trends, customer behavior, and key business insights. It includes data cleaning, exploratory data analysis, and an interactive dashboard to support data-driven decision-making.

## Key Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- GitHub

## Key Insights
- Consumer segment has highest sales
- West region performs best
- Sales show seasonal trends
- Few products drive most sales
- High discounts reduce profit
- Some regions have low profit
